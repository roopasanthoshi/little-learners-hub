# Little Learners Hub

An educational web platform designed to teach children moral values, social ethics, gender equality, and personality development.

## Features
- Interactive educational interface
- Bootstrap responsive design
- Embedded learning videos
- Carousel banner section
- Core values showcase
- Child-friendly UI

## Tech Stack
- HTML
- CSS
- Bootstrap
